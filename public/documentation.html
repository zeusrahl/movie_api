Documentation | Movie API

Intro Paragraph

This project was to build an API to help organize a set of data about movies. Mainly, it will revolve around users interacting movie names, directors, and genres. Additionally, users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request Method URL Response
List all movies to users GET /movies List of all available movies with image and title
Return data of requested movie GET /movies/[movie title] A JSON object with movie title, cover art, description, genre, featured film, director and actors
List of movies by genre GET /movies/[genre] Returns a list of movies that satisfy the requested genre
Return director information GET /directors/[director name] Gives information about requested director with respect to their bio, birth day, death year (if deceased), and featured work
Register a new user POST /users Add a new user to be registered
Update a username PUT /users/[user-name] Update the username of a user
Users add movie to their favorites List POST /users/[user-name]/movies/[movieID] Add a movie to User's list of movies
Delete movie from user's List DELETE /users/[user-name]/movies/[movieID]
Deregister an account DELETE /users/[user-name] Username will be deleted from data
